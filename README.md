# Сочинения ЕГЭ — PWA версия

## Что изменилось
Electron → PWA. Все данные хранятся в localStorage браузера (сохраняются между сессиями).

## Как установить на iPhone

1. **Разместите файлы** на любом хостинге с HTTPS:
   - GitHub Pages (бесплатно): загрузите папку в репозиторий → Settings → Pages
   - Netlify (бесплатно): перетащите папку на netlify.com/drop
   - Vercel, любой другой хостинг

2. **Откройте сайт в Safari** на iPhone (именно Safari, не Chrome)

3. Нажмите кнопку **«Поделиться»** (квадрат со стрелкой вверх)

4. Выберите **«На экран "Домой"»**

5. Нажмите **«Добавить»**

Готово — приложение появится на рабочем столе как обычная иконка и будет работать без интернета.

## Файлы проекта
- `index.html` — разметка
- `style.css` — стили (тёмная тема, адаптирована под iPhone)
- `app.js` — логика (замена Electron IPC → localStorage)
- `manifest.json` — метаданные PWA
- `sw.js` — Service Worker (офлайн-режим)
